# TURBOTYPE

Professional Speed Typing Test Application
*Web Programming Project 2026*

TurboType is a high-performance web application designed to measure and improve typing speed and accuracy. Built with a focus on minimalist design and real-time feedback, it offers a seamless experience for developers and typing enthusiasts.


Key Features :

Real-time Analytics
Calculate your Words Per Minute (WPM) and Accuracy percentage as the user is typing.

Live Error Tracking
Visual feedback for mistakes with a dedicated error counter on the main dashboard.

Dynamic Progress Bar
A smooth, visual countdown representing your remaining test time.

Flexible Durations
Choose between 15s, 30s, or 60s test modes to challenge your speed and practice at your own pace.

Elite Leaderboard
Persistent Top 5 Best Scores storage in Leaderboard.Your progress is saved even after refreshing the page.

Technical Stack:

HTML5 : Semantic structure for optimal performance and accessibility.
CSS3m : Custom properties, Flexbox layout, and smooth transitions for a premium feel.
JavaScript ES6 : Advanced DOM manipulation, asynchronous timers, and JSON-based data persistence.


Working principles : 

1. State Management
Tracking user progress and game status in real-time.
2. Data Persistence
Utilizing JSON to manage a local database of high scores.
3. Event Handling
High-frequency input listeners for instantaneous validation.
4. Asynchronous Programming
Managing test intervals and UI updates via the JavaScript Event Loop.
